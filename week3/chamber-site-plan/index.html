Purpose

Site Name and Domain

The site is going to be called WDD Chamber.of Commerce. The domain is: wddchamber.com.

Site Purpose

WDD Chamber of Commerce will be a communtiy page to promote the business interests of the WDD community. It will contain the following pages: Home, Discover, Directory, Join and Contact. The home page will have pictures, community information and weather. Discover will be about the rich heritage and history of the area, as well as current events. Directory will list local businesses. Join is where you will find information about membership and an application form. Contact will contain contact information for the chamber and its officers. It will also have a contact form for submitting messages to the chamber.

The secondary purpose of this page is to be a student learning project for WDD230.
